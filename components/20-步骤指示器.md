### 20. 步骤指示器（替换"第 X 步"文字）

> **🎯 效果**：水平步骤条，已完成实心圆（主题色）+ 白色数字，进行中描边圆 + 发光环，待办灰色描边。圆点间灰色连接线贯通。窄屏转为横向可滚动。

当需要引导读者了解流程进度时，用水平步骤条替代"第一步/第二步/第三步"的文字表述。

### HTML

```html
<div class="step-indicator" data-anim="fade-up">
  <div class="si-step done"><div class="si-num">1</div><div class="si-label">需求分析</div></div>
  <div class="si-step done"><div class="si-num">2</div><div class="si-label">系统设计</div></div>
  <div class="si-step current"><div class="si-num">3</div><div class="si-label">开发实施</div></div>
  <div class="si-step"><div class="si-num">4</div><div class="si-label">测试验收</div></div>
  <div class="si-step"><div class="si-num">5</div><div class="si-label">部署上线</div></div>
</div>
```

### 状态语义

| 状态 | 圆点 | 说明 |
|------|------|------|
| `done` | 实心主题色 + 白色数字 | 已完成 |
| `current` | 白色描边 + 主题色数字 + 发光环 | 进行中 |
| 无 class | 灰色描边 + 灰色数字 | 待办 |

### 使用规则

- 每链 3-6 步，超过拆为多个
- 标签 ≤4 字
- 至少有一个 `done` 或 `current`

### 降级说明

- **窄屏（<500px）**：连接线隐藏，横向可滚动
- **不支持 `::before`**：连接线消失，步骤独立展示
