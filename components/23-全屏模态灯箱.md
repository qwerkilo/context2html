### 23. 全屏模态/灯箱（替换图片外链）

当需要放大查看图表、示意图或展示详情时，用灯箱替代跳转到新页面或打开大图链接。点击触发元素，全屏遮罩层出现，内容居中放大显示。支持 ESC 关闭、点击遮罩关闭。

HTML 结构（触发元素 + 灯箱面板）：
```html
<!-- 触发方式 1：按钮触发 -->
<button class="lbox-trigger" data-lbox="chart-1"><svg viewBox="0 0 20 20" width="16" height="16" fill="none" stroke="currentColor" stroke-width="1.5" stroke-linecap="round"><circle cx="8.5" cy="8.5" r="4.5"/><path d="M12 12l4 4"/></svg> 点击放大</button>

<!-- 触发方式 2：图片触发 -->
<img src="chart-thumb.png" class="lbox-trigger" data-lbox="chart-1" alt="图表缩略图" style="cursor:pointer;">

<!-- 灯箱面板 -->
<div class="lbox" id="lbox-chart-1">
  <div class="lbox-overlay"></div>
  <div class="lbox-content">
    <button class="lbox-close"><svg viewBox="0 0 20 20" width="16" height="16" fill="none" stroke="currentColor" stroke-width="1.5" stroke-linecap="round"><path d="M5 5l10 10M15 5l-10 10"/></svg></button>
    <img src="chart-full.png" alt="图表大图">
  </div>
</div>
```

使用规则：
- 一个页面最多 3 个灯箱，过多影响性能
- 灯箱内可放图片或 HTML 内容（替换 `<img>` 为 `<div>` 容器）
- 触发按钮文字控制在 10 字以内
- 缩略图和原图宽高比一致，避免视觉跳跃

CSS：
```css
.lbox { position: fixed; inset: 0; z-index: 1001; display: flex; align-items: center; justify-content: center; visibility: hidden; }
.lbox.open { visibility: visible; }
.lbox-overlay { position: absolute; inset: 0; background: rgba(0,0,0,0.8); opacity: 0; transition: opacity 0.3s ease; }
.lbox.open .lbox-overlay { opacity: 1; }
.lbox-content { position: relative; max-width: 90vw; max-height: 90vh; transform: scale(0.92); transition: transform 0.3s ease; }
.lbox.open .lbox-content { transform: scale(1); }
.lbox-close { position: absolute; top: -2.2em; right: 0; background: none; border: none; color: #fff; font-size: 1.5rem; cursor: pointer; padding: 0.2em; line-height: 1; opacity: 0.8; }
.lbox-close:hover { opacity: 1; }
.lbox-close:active { transform: scale(0.98); }
.lbox-close:focus-visible { outline: 2px solid var(--accent); outline-offset: 2px; border-radius: 2px; }
.lbox-trigger { cursor: pointer; }
.lbox-trigger:active { transform: scale(0.98); }
.lbox-trigger:focus-visible { outline: 2px solid var(--accent); outline-offset: 2px; }
.lbox-content img { display: block; max-width: 100%; max-height: 85vh; border-radius: 4px; box-shadow: 0 4px 20px rgba(0,0,0,0.3); }
```

JS：
```html
<script>
// 全屏模态/灯箱
(function(){document.querySelectorAll('[data-lbox]').forEach(function(t){t.addEventListener('click',function(){
var id=this.dataset.lbox,lb=document.getElementById('lbox-'+id);if(lb){lb.classList.add('open');document.body.style.overflow='hidden';}});});
document.querySelectorAll('.lbox-close,.lbox-overlay').forEach(function(e){e.addEventListener('click',function(){
var lb=this.closest('.lbox');lb.classList.remove('open');document.body.style.overflow='';});});
document.addEventListener('keydown',function(e){if(e.key==='Escape'){document.querySelectorAll('.lbox.open').forEach(function(lb){lb.classList.remove('open');document.body.style.overflow='';});}});})();
</script>
```
