### 2. 角色卡片（替换文本列表）

当需要介绍多个参与者或对比多方观点时，用 SVG 卡片网格。3×2 网格，每张卡片 210×100，圆角 8px，带阴影。

CSS：
```css
.role-grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 1rem; max-width: 700px; margin: 1.5rem auto; }
.role-card { background: var(--bg); border: 1px solid var(--border); border-radius: 8px; padding: 0; overflow: hidden; box-shadow: 0 2px 8px rgba(0,0,0,0.08); }
.role-header { background: var(--accent); color: var(--accent-text); padding: 6px 10px; font-size: 0.78rem; font-weight: 600; display: flex; align-items: center; gap: 6px; }
.role-body { padding: 10px; font-size: 0.82rem; color: var(--text); line-height: 1.5; }
.role-body p { margin: 0 0 4px 0; color: var(--muted); font-size: 0.78rem; }
@media (max-width: 600px) { .role-grid { grid-template-columns: repeat(2, 1fr); } }
```

HTML：
```html
<div class="role-grid">
  <div class="role-card">
    <div class="role-header" style="background:#3b82f6">🏛️ 决策者</div>
    <div class="role-body">负责战略方向与投资决策<p>关注 ROI 和周期</p></div>
  </div>
  <div class="role-card">
    <div class="role-header" style="background:#f59e0b">⚡ 执行者</div>
    <div class="role-body">负责落地实施与日常运营<p>关注效率和成本</p></div>
  </div>
  <div class="role-card">
    <div class="role-header" style="background:#ef4444">⚠️ 监督者</div>
    <div class="role-body">负责合规审查与风险控制<p>关注合规和安全</p></div>
  </div>
</div>
```

使用规则：
- 每张卡片最多 3 行描述，单行不超过 25 字
- 3×2 网格适合介绍 4-6 个角色，少于 4 个用 2×2
- 卡片顶栏颜色按角色类型区分（蓝=正面、橙=触发者、红=负面、绿=救助方）

降级说明：
- **SVG 中文字体不渲染**：确认 `<style>` 内 font-family 包含中文字体（"Noto Sans CJK SC"）
- **卡片溢出容器**：给外部 `<svg>` 加 `max-width: 100%; height: auto;`
