### 5. 对比表（基础版）

当需要对比少量条目（≤5 行 × ≤6 列）时使用基础 HTML 表格。超过此规模请用增强版组件 #22。

CSS：
```css
.cmp-table { width: 100%; border-collapse: collapse; margin: 1.5rem 0; font-size: 0.88rem; }
.cmp-table th { background: color-mix(in srgb, var(--accent) 8%, var(--bg)); color: var(--accent); font-weight: 600; padding: 0.6em 0.8em; text-align: left; white-space: nowrap; }
.cmp-table td { padding: 0.5em 0.8em; border-bottom: 1px solid var(--border); }
.cmp-table tr:hover td { background: color-mix(in srgb, var(--accent) 3%, var(--bg)); }
.cmp-caption { font-size: 0.8rem; color: var(--muted); margin-top: -0.8rem; text-align: center; }
@media (max-width: 700px) {
  .cmp-table { font-size: 0.78rem; }
  .cmp-table thead { display: none; }
  .cmp-table tr { display: block; margin-bottom: 0.6em; border: 1px solid var(--border); border-radius: 6px; padding: 0.4em; }
  .cmp-table td { display: flex; justify-content: space-between; align-items: center; padding: 0.4em 0.5em; border-bottom: 1px solid var(--border); }
  .cmp-table td:last-child { border-bottom: none; }
  .cmp-table td::before { content: attr(data-label); font-weight: 600; color: var(--muted); font-size: 0.75rem; margin-right: 0.5em; }
}
```

HTML：
```html
<table class="cmp-table">
  <thead><tr><th></th><th>方案 A</th><th>方案 B</th></tr></thead>
  <tbody>
    <tr><td data-label="维度">成本</td><td data-label="方案 A">低</td><td data-label="方案 B">高</td></tr>
    <tr><td data-label="维度">速度</td><td data-label="方案 A">快</td><td data-label="方案 B">中</td></tr>
    <tr><td data-label="维度">维护</td><td data-label="方案 A">易</td><td data-label="方案 B">中</td></tr>
  </tbody>
</table>
<p class="cmp-caption">数据来源：团队评估（2026）</p>
```

> 如需固定表头、列排序、斑马纹、悬停高亮等高级功能，请直接使用 [#22 对比表增强版](22-对比表增强版.md)。
