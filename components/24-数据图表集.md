### 24. 数据图表集（柱状图 / 饼图 / 折线图 / 堆叠图）

纯 CSS/SVG 数据图表，无 JS 绘图依赖。四种图表类型覆盖常见数据展示需求，全部使用 `var()` 变量跟随主题切换。

#### 24a. 纵向柱状图（比较分类数值）

适合比较多个类别的数值大小，与 #4 水平条形图互补。

HTML 结构：
```html
<div class="vchart" data-anim="fade-up">
  <div class="vc-group">
    <div class="vc-bar" style="--h:85;" data-label="美国"><span class="vc-val">7,000亿</span></div>
    <div class="vc-bar" style="--h:65;" data-label="中国"><span class="vc-val">5,860亿</span></div>
    <div class="vc-bar" style="--h:45;" data-label="日本"><span class="vc-val">3,300亿</span></div>
    <div class="vc-bar" style="--h:30;" data-label="英国"><span class="vc-val">2,200亿</span></div>
  </div>
</div>
```

使用规则：
- `--h` 为百分比高度（0-100），最大值对应最高柱
- 柱状图 3-8 根柱子为宜
- 数值标签显示在柱顶

CSS：
```css
.vchart { margin: 2rem 0; overflow-x: auto; }
.vc-group { display: flex; align-items: flex-end; gap: 0.6em; height: 200px; padding: 0 0.5em 2em 0.5em; border-bottom: 1px solid var(--border); }
.vc-bar { flex: 1; display: flex; flex-direction: column; align-items: center; justify-content: flex-start; height: calc(var(--h) * 1%); background: var(--accent); border-radius: var(--radius, 4px) var(--radius, 4px) 0 0; min-width: 36px; position: relative; transition: height 0.3s ease-out; }
.vc-bar::after { content: attr(data-label); position: absolute; bottom: -1.8em; left: 50%; transform: translateX(-50%); font-size: 0.75rem; color: var(--muted); white-space: nowrap; }
.vc-val { position: absolute; top: -1.5em; font-size: 0.72rem; font-weight: 600; color: var(--text); white-space: nowrap; font-variant-numeric: tabular-nums; }
```

#### 24b. 饼图/环形图（占比展示）

适合展示各部分占总体的比例。使用 SVG `stroke-dasharray` 实现。

HTML 结构：
```html
<figure class="svg-fig" data-anim="fade-up">
  <svg viewBox="0 0 100 100" class="pie-chart">
    <circle cx="50" cy="50" r="40" fill="none" stroke="var(--border)" stroke-width="20" opacity="0.3"/>
    <circle cx="50" cy="50" r="40" fill="none" stroke="var(--accent)" stroke-width="20" stroke-dasharray="88 163" stroke-dashoffset="0" transform="rotate(-90 50 50)"/>
    <circle cx="50" cy="50" r="40" fill="none" stroke="var(--success)" stroke-width="20" stroke-dasharray="63 189" stroke-dashoffset="-88" transform="rotate(-90 50 50)"/>
    <circle cx="50" cy="50" r="40" fill="none" stroke="var(--warning)" stroke-width="20" stroke-dasharray="50 201" stroke-dashoffset="-151" transform="rotate(-90 50 50)"/>
    <circle cx="50" cy="50" r="40" fill="none" stroke="var(--error)" stroke-width="20" stroke-dasharray="30 221" stroke-dashoffset="-201" transform="rotate(-90 50 50)"/>
  </svg>
  <figcaption class="pie-legend">
    <span class="pl-item"><span class="pl-dot" style="background:var(--accent);"></span>消费 35%</span>
    <span class="pl-item"><span class="pl-dot" style="background:var(--success);"></span>投资 25%</span>
    <span class="pl-item"><span class="pl-dot" style="background:var(--warning);"></span>政府支出 20%</span>
    <span class="pl-item"><span class="pl-dot" style="background:var(--error);"></span>出口 12%</span>
  </figcaption>
</figure>
```

使用规则：
- 饼图切片不超过 6 块，超过则合并为"其他"
- 数值总计 100%
- 颜色语义：最大块用 `--accent`，其余依次使用语义色

CSS：
```css
.pie-chart { max-width: 240px; display: block; margin: 0 auto; }
.pie-legend { display: flex; flex-wrap: wrap; justify-content: center; gap: 0.4em 1em; margin-top: 0.5em; font-size: 0.8rem; }
.pl-item { display: flex; align-items: center; gap: 0.3em; font-variant-numeric: tabular-nums; }
.pl-dot { width: 10px; height: 10px; border-radius: 50%; flex-shrink: 0; }
```

#### 24c. 折线图（趋势变化）

适合展示随时间变化的趋势。

HTML 结构：
```html
<figure class="svg-fig" data-anim="fade-up">
  <svg viewBox="0 0 300 160" class="line-chart">
    <!-- 网格线 -->
    <line x1="0" y1="120" x2="300" y2="120" stroke="var(--border)" stroke-width="1" stroke-dasharray="4 4"/>
    <line x1="0" y1="80" x2="300" y2="80" stroke="var(--border)" stroke-width="1" stroke-dasharray="4 4"/>
    <line x1="0" y1="40" x2="300" y2="40" stroke="var(--border)" stroke-width="1" stroke-dasharray="4 4"/>
    <!-- 折线 -->
    <polyline points="0,140 50,120 100,130 150,90 200,100 250,50 300,60" fill="none" stroke="var(--accent)" stroke-width="2.5" stroke-linejoin="round" stroke-linecap="round"/>
    <!-- 数据点 -->
    <circle cx="0" cy="140" r="4" fill="var(--accent)" stroke="var(--bg)" stroke-width="1.5"/>
    <circle cx="50" cy="120" r="4" fill="var(--accent)" stroke="var(--bg)" stroke-width="1.5"/>
    <circle cx="100" cy="130" r="4" fill="var(--accent)" stroke="var(--bg)" stroke-width="1.5"/>
    <circle cx="150" cy="90" r="4" fill="var(--accent)" stroke="var(--bg)" stroke-width="1.5"/>
    <circle cx="200" cy="100" r="4" fill="var(--accent)" stroke="var(--bg)" stroke-width="1.5"/>
    <circle cx="250" cy="50" r="4" fill="var(--accent)" stroke="var(--bg)" stroke-width="1.5"/>
    <circle cx="300" cy="60" r="4" fill="var(--accent)" stroke="var(--bg)" stroke-width="1.5"/>
  </svg>
</figure>
```

使用规则：
- 数据点 5-10 个为宜
- Y 轴范围自动适应数据（手动调整 viewBox 高度）
- 网格线用虚线，不抢眼

CSS：
```css
.line-chart { max-width: 100%; height: auto; }
.line-chart line { stroke: var(--border); }
```

#### 24d. 堆叠条形图（部分与整体）

适合展示各部分的构成以及总和对比。每个水平条内分段。

HTML 结构：
```html
<div class="stack-chart" data-anim="fade-up">
  <div class="sc-row">
    <div class="sc-label">美国</div>
    <div class="sc-track">
      <div class="sc-seg" style="width:45%; background:var(--accent);" title="消费 45%"></div>
      <div class="sc-seg" style="width:25%; background:var(--success);" title="投资 25%"></div>
      <div class="sc-seg" style="width:15%; background:var(--warning);" title="政府 15%"></div>
      <div class="sc-seg" style="width:10%; background:var(--error);" title="出口 10%"></div>
      <div class="sc-seg" style="width:5%; background:var(--border);" title="其他 5%"></div>
    </div>
  </div>
  <div class="sc-row">
    <div class="sc-label">中国</div>
    <div class="sc-track">
      <div class="sc-seg" style="width:38%; background:var(--accent);"></div>
      <div class="sc-seg" style="width:30%; background:var(--success);"></div>
      <div class="sc-seg" style="width:18%; background:var(--warning);"></div>
      <div class="sc-seg" style="width:14%; background:var(--error);"></div>
    </div>
  </div>
</div>
```

使用规则：
- 每行分段 3-6 段
- 各段宽度百分比之和 = 100%
- 第一段用 `--accent`，其余按顺序用语义色

CSS：
```css
.stack-chart { margin: 1.5rem 0; }
.sc-row { display: flex; align-items: center; margin: 0.4em 0; gap: 0.8em; }
.sc-label { width: 60px; flex-shrink: 0; font-size: 0.82rem; font-weight: 500; color: var(--text); text-align: right; }
.sc-track { flex: 1; display: flex; height: 1.5em; border-radius: 4px; overflow: hidden; }
.sc-seg { height: 100%; transition: width 0.3s ease-out; }
.sc-seg:first-child { border-radius: 4px 0 0 4px; }
.sc-seg:last-child { border-radius: 0 4px 4px 0; }
```

降级说明（所有图表共用）：
- **不支持 CSS `var()`**：颜色降级为默认值
- **SVG 不支持**：饼图/折线图退化为文字描述
- **窄屏**：柱状图和堆叠图通过 `overflow-x: auto` 支持横向滚动
