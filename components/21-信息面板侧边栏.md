### 21. 信息面板/侧边栏（替换脚注/附录）

> **🎯 效果**：右侧抽屉式面板（380px），滑入/滑出动画 0.3s ease。半透明遮罩层 + 粘性标题栏 + 关闭按钮。支持 ESC 关闭、点击遮罩关闭。触发按钮虚线边框 + 主题色。

当正文中需要提供扩展阅读、人物简介、术语详解等补充信息时，用从右侧滑入的面板替代脚注或附录跳转。

### HTML

```html
<p>正文内容……<button class="ip-btn" data-panel="glossary-mmt">
  <svg viewBox="0 0 20 20" width="16" height="16" fill="none" stroke="currentColor" stroke-width="1.5" stroke-linecap="round"><circle cx="10" cy="10" r="8"/><path d="M10 9v5M10 7v.01"/></svg> MMT 详解
</button></p>

<div class="info-panel" id="panel-glossary-mmt">
  <div class="ip-overlay"></div>
  <div class="ip-drawer">
    <div class="ip-header">
      <h4>现代货币理论（MMT）</h4>
      <button class="ip-close"><svg viewBox="0 0 20 20" width="16" height="16" fill="none" stroke="currentColor" stroke-width="1.5" stroke-linecap="round"><path d="M5 5l10 10M15 5l-10 10"/></svg></button>
    </div>
    <div class="ip-body"><p>详细说明内容……</p></div>
  </div>
</div>
```

### JS

```html
<script>
(function(){document.querySelectorAll('.ip-btn').forEach(function(b){b.addEventListener('click',function(){
var p=document.getElementById('panel-'+this.dataset.panel);if(p){p.classList.add('open');document.body.style.overflow='hidden';}});});
document.querySelectorAll('.ip-close,.ip-overlay').forEach(function(e){e.addEventListener('click',function(){
var p=this.closest('.info-panel');p.classList.remove('open');document.body.style.overflow='';});});
document.addEventListener('keydown',function(e){if(e.key==='Escape'){document.querySelectorAll('.info-panel.open').forEach(function(p){p.classList.remove('open');document.body.style.overflow='';});}});})();
</script>
```

### 使用规则

- 一课最多 3 个信息面板
- 面板内容控制在屏幕 70% 高度以内（可滚动）
- 触发按钮文字 ≤10 字
