### 12. 数据卡片网格（替换文本数据段）

> **🎯 效果**：CSS Grid 自适应卡片网格，每张卡片包含 emoji 图标 + 大号强调色数值 + 单位 + 标签。悬停时上浮 2px + 阴影加深。多张卡片依次延迟 50ms 错开悬停反馈。

当需要展示一组关键统计数据时，用卡片网格替代散落在段落中的数字。

### HTML

```html
<div class="data-grid">
  <div class="dg-card" data-anim="fade-up">
    <div class="dg-icon">📉</div>
    <div class="dg-value">8.8</div>
    <div class="dg-unit">万亿美元</div>
    <div class="dg-label">全球财富蒸发</div>
  </div>
  <div class="dg-card" data-anim="fade-up">
    <div class="dg-icon">🏦</div>
    <div class="dg-value">127</div>
    <div class="dg-unit">家</div>
    <div class="dg-label">倒闭银行数</div>
  </div>
  <div class="dg-card" data-anim="fade-up">
    <div class="dg-icon">🌍</div>
    <div class="dg-value">0.1</div>
    <div class="dg-unit">%</div>
    <div class="dg-label">全球 GDP 萎缩</div>
  </div>
</div>
```

### CSS

```css
.data-grid {
  display: grid; grid-template-columns: repeat(auto-fill, minmax(180px, 1fr));
  gap: 1rem; margin: 1.5rem 0;
}
.dg-card {
  background: var(--bg); border: 1px solid var(--border);
  border-radius: var(--radius, 8px); padding: 1.2em 0.8em; text-align: center;
  cursor: default;
  transition: transform 0.25s ease, box-shadow 0.25s ease;
}
.dg-card:hover { transform: translateY(-3px); box-shadow: 0 6px 20px rgba(0,0,0,0.1); }
.dg-card:nth-child(2) { transition-delay: 50ms; }
.dg-card:nth-child(3) { transition-delay: 100ms; }
.dg-card:nth-child(4) { transition-delay: 150ms; }
.dg-card:nth-child(5) { transition-delay: 200ms; }
.dg-card:nth-child(6) { transition-delay: 250ms; }
.dg-card:nth-child(7) { transition-delay: 300ms; }
.dg-icon { font-size: 2rem; margin-bottom: 0.25em; line-height: 1; }
.dg-value {
  font-size: 1.8rem; font-weight: 800; color: var(--accent);
  line-height: 1.1; font-variant-numeric: tabular-nums;
}
.dg-unit { font-size: 0.8rem; color: var(--accent); font-weight: 500; margin-bottom: 0.5em; }
.dg-label { font-size: 0.85rem; color: var(--muted); line-height: 1.3; }
```

### 布局参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 卡片数 | 3-8 张 | 超过拆分 |
| 最小列宽 | 180px | CSS Grid `auto-fill` |
| 数值 | ≤5 位数字 | 超过用万/亿缩写 |
| 悬停 | 上浮 3px + 阴影加深 | 0.25s ease |

### 使用规则

- `dg-icon` 使用 emoji，不要用图片
- 数值一般不超过 5 位数字，标签不超过 10 字
- JS：不需要，入场动画复用 `data-anim="fade-up"`

### 降级说明

- **不支持 CSS Grid**（IE11）：退化为全宽堆叠显示
- **不支持 IntersectionObserver**：卡片直接可见，无动画
