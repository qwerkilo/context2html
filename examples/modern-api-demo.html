现代浏览器 API 组件 — 示例

25a. 原生折叠（<details>）

零 JS 的折叠组件。点击展开/收起，浏览器原生支持。

1 泡沫积累期

2001-2006 年，美国房地产泡沫迅速膨胀。低利率环境刺激了信贷扩张，次级抵押贷款规模从 1000 亿美元增长到 6000 亿美元。金融机构大量发行 MBS 和 CDO，风险在全球范围内分散。

2 危机爆发期

2007 年房价见顶后开始下跌，次贷违约率飙升。2008 年 3 月贝尔斯登被收购，9 月雷曼兄弟破产，全球信贷市场冻结。恐慌从华尔街蔓延至实体经济。

3 政策应对期

各国政府大规模干预：美国 TARP 7000 亿美元、美联储零利率+QE、中国 4 万亿刺激。到 2009 年底全球经济止跌回升。

25d. Popover 提示

点击按钮弹出轻量提示，点击外部自动关闭。零 JS。

在金融危机研究中， 是一个核心概念——指金融机构因为知道政府会救助而承担过度风险。

道德风险（Moral Hazard）： 当一方在某种程度上受到保护而无需承担风险后果时，倾向于做出比正常情况下更冒险的行为。
