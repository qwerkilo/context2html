系统架构示意图（替换为真实图片）
前端入口层：用户通过浏览器或移动端 App 访问系统。CDN 加速静态资源分发，负载均衡器将请求分发到下游服务。
API 网关层：统一入口，处理认证鉴权、限流熔断、请求路由。将请求转发到对应的微服务集群。
微服务集群：按业务域拆分的多个独立服务（用户服务、订单服务、支付服务等），每个服务可独立部署和扩缩容。
数据存储层：关系型数据库（MySQL）存储核心业务数据，Redis 缓存热点数据，Elasticsearch 提供全文搜索能力。
消息队列与监控：Kafka/RabbitMQ 解耦异步任务，Prometheus + Grafana 监控系统状态，ELK 集中管理日志。
